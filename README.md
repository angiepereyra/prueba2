Aplicación Fullstack

Tecnologías:

- Backend: Spring Boot, Java 17, Maven, elaborado en Intellij
-> Ejecutar el códgo http://localhost:8080
Tengo experiencia laboral en Java 6 nativo en compañías de seguros; sin embargo,he ido actualizándome realizando proyectos y dictando cursos de Spring Boot y AWS en la facultad de Informática de la PUCP, mi universidad.
Se utilizó 1 DTO para los datos que se reciben de la API externa de cryptos y una entidad para guardar los datos en la bd y mostrarlo en el front.
El controller requiere hacer un GET a la API externa de cryptos, POSTear los favoritos y otro GET para las alertas; el service hace la lógica de ir al API externa y transformar los datos. Aparte el repsitorio JPA para que sea más sencillo

- Frontend: React Vite, Bootstrap 5, Node.js, elaborado en Visual Studio Code
-> Ejecutar con npm run dev http://localhost:5173
Para complementar mis conocimientos en backend aprendí a usar React, el estilo usado para esta prueba lo he utilizado antes en un proyecto de una página web calendario de los salones de las facultades de ingeniería (colores principales de la facultad morado y verde), por ello también el título utiliza une fuente de Google Fonts, que se asemeja a una letra pixeleada o digital. En esta parte lo más complicado fue la lógica ya que se te restringe el acceso a la API si la llamas muchas veces
